# A-small-toy
一款 ASCII 艺术字时钟，自动实时刷新并显示系统日期与时间，采用终端风格 GUI 界面呈现。

## 📥 下载可直接运行的程序
如需**无需安装环境、可以直接运行**的二进制可执行文件（Windows / macOS / Linux），请前往项目 **Releases** 页面下载：
👉 [前往 Releases 下载](https://github.com/LhyYBMQ520/A-small-toy/releases)

## 特色
以 Python 版本为基准实现，我们将使用多种编程语言复刻同款功能。

## 项目结构
所有语言实现均存放于 `src/` 目录下，按语言与开发者 ID 区分。
